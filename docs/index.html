How it works

02

03

Install

Two ways to add Obsidian Agent to your vault.

Manual install

  1. Download the latest release assets.
  2. Place main.js, manifest.json, and styles.css in .obsidian/plugins/obsidian-agent/.
  3. Reload Obsidian and enable the plugin from Settings → Community plugins.

Development setup

  1. Clone the repository and run npm install.
  2. Run npm run dev for watch mode.
  3. Use npm run build and npm run lint before packaging a release.

FAQ

Questions you will probably ask first.

Does it work only with cloud providers?

No. You can point it at custom OpenAI-compatible or Anthropic-compatible endpoints when available.

Can it modify my files automatically?

Agent mode can perform vault operations, and write-like actions can require explicit confirmation.

What data is stored locally?

Plugin settings and chat history are persisted through Obsidian plugin data APIs.
